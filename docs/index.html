IBM Cloud CLI AI

An AI-powered assistant for the IBM Cloud CLI that translates natural language queries into IBM Cloud commands.

Natural Language Processing

Convert plain English requests into IBM Cloud CLI commands without memorizing complex syntax.

Interactive Chat Mode

Engage in a conversation-like interface with the CLI for a more intuitive experience.

Command Editing

Review and modify translated commands before execution to ensure they match your intent.

Command Execution

Run the commands directly from the interface with a simple confirmation.

Installation

  1. Clone the repository: 
    git clone git@github.ibm.com:watsonx-apac/ibmcloud-cli-ai.git
  2. Create a .env file with your WatsonX API credentials: 
    WATSONX_API_KEY=your_api_key
WATSONX_PROJECT_ID=your_project_id
  3. Build the project: 
    cargo build

Usage

Ask a single question

cargo run -- ask "list all my code engine instances"

Start interactive chat mode

cargo run -- chat

In chat mode:

  • Type your query in natural language
  • The AI will translate it to an IBM Cloud command
  • Edit the command if needed or press Enter to execute
  • Type exit or quit to end the session
