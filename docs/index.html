Installation

  1. Clone the repository: 
    git clone git@github.ibm.com:watsonx-apac/ibmcloud-cli-ai.git
  2. Create a .env file with your WatsonX API credentials: 
    WATSONX_API_KEY=your_api_key
WATSONX_PROJECT_ID=your_project_id
  3. Build the project: 
    cargo build

Usage

Ask a single question

cargo run -- ask "list all my code engine instances"

Start interactive chat mode

cargo run -- chat

In chat mode:

  • Type your query in natural language
  • The AI will translate it to an IBM Cloud command
  • Edit the command if needed or press Enter to execute
  • Type exit or quit to end the session

Examples

Natural Language to IBM Cloud Commands

Here are some examples of natural language queries and their translated IBM Cloud commands:

Query: "Show me all my code engine applications"
Command: ibmcloud ce app list
Query: "List my Watson Machine Learning instances"
Command: ibmcloud resource service-instances --service-name watson-machine-learning
Query: "Create a new VPC in us-south"
Command: ibmcloud is vpc-create my-vpc --zone us-south-1
